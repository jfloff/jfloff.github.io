Coding

Vadara

February 2014 - Present

Vadara is a totally generic framework regarding the employed elasticity strategy; that bypasses Cloud Provider (CP) elasticity lock-in, allowing the development of elasticity strategies that are not tied with any CP; that abstracts CPs APIs and design specificities thereby decoupling elasticity from any underlying infrastructure; that allows a cloud application to be placed on multiple CP at the same time; and that takes full-period and instance startup time issues into consideration. This project results from my Master Thesis research.
